## What this is

A reusable analytical brief format for the Operations Latam team. Three components:

1. **Artifact template** — .docx with placeholder text and inline methodology notes
2. **Methodology playbook** — bottom-up + sensitivity + scenarios approach; generic for any operational analysis
3. **Example library** — funnel analysis as Exhibit A; build more over time

---

## Section structure

- Header block (title, subtitle, author, date, shared-with, supporting model reference)
- Executive summary (2–4 sentences — thesis, magnitude, consequence)
- Current state (metrics only needed to understand the problem)
- Diagnostic (the "why" section)
- Findings / coordinated path (table: dimension / current / target / lever)
- Additional context (secondary finding or qualifier — optional)
- Proposed discussion points (peer-to-peer framing, not "asks")
- Footer (supporting materials reference)
